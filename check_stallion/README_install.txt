Install
-------

To create a Python's 'virtual environment' based on Python 3.9.

For example, using 'Anaconda Python Distribution' it is possible to use

    conda create -n stallion python=3.9

To activate the new 'virtual environment'

    conda activate stallion

To install the 3rd parts libraries contained in 'requirements.txt', using 'pip':

    pip install -r requirements.txt

Now the environment and all libraries are installed


Run the example
---------------

    python check_rnn_cnn.py
